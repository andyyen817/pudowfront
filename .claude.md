# Claude 开发规范

## 代码规范
- 所有代码、变量名、函数名、注释必须使用英文
- 代码风格遵循项目现有规范
- 使用清晰的命名，避免缩写

## 交流规范
- 所有回复和解释使用中文
- 技术术语可以保留英文原文
- 确保沟通清晰准确

## 示例

### ✅ 正确示例

```javascript
// Good: Code in English
const handleSubmit = (event) => {
  event.preventDefault();
  console.log('Form submitted');
};
```

回复：我已经创建了表单提交处理函数 `handleSubmit`，它会阻止默认行为并记录提交日志。

### ❌ 错误示例

```javascript
// Bad: Code in Chinese
const 处理提交 = (事件) => {
  事件.preventDefault();
  console.log('表单已提交');
};
```

回复：I have created the form submission handler function.
